# 🌟 StarsGallery

Une application Android permettant de découvrir et d'évaluer vos stars préférées. L'application offre une expérience utilisateur fluide avec des animations, un filtrage en temps réel et la possibilité de modifier les notes des acteurs et actrices.

## 📱 Fonctionnalités

- ✅ **Splash Screen Animé** - Animation d'accueil avec rotation, zoom, translation et fondu
- ✅ **Liste des Stars** - Affichage des stars avec RecyclerView et images circulaires
- ✅ **Filtrage Dynamique** - Recherche en temps réel avec SearchView
- ✅ **Modification des Notes** - Popup personnalisé pour modifier la note d'une star
- ✅ **Partage** - Partage de l'application via les applications installées
- ✅ **Chargement d'Images** - Utilisation de Glide pour charger les images distantes
